# Hexlet PHP console projects

![CI](https://github.com/divivaes/php-project-lvl1/workflows/CI/badge.svg)

## Project setup guide

### Globally

```bash
$ composer global require divivaes/php-project-lvl1
```

```bash
$ cd $HOME/.composer/vendor/divivaes/php-project-lvl1
```

### Project run command

```bash
$ php bin/brain-even 
```

```bash
$ php bin/brain-calc 
```

```bash
$ php bin/brain-gcd
```
